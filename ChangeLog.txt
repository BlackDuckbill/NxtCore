NxtCore Change Log
==================

Version 2.0.0
  - New JSON support
  - New fields for Nxt.getBlockchainStatus()
  - Nxt.getInboundPeers() added

Version 1.6.0
  - New transaction types for NRS 1.5
  - Support HTTP response compression
  - Support 'addEvents' and 'removeEvents' parameters for eventRegister API
  - Support 'includeTransactions' parameter for getBlock API
  - Support 'includePeerInfo' parameter for getPeers API
  - Nxt.addPeer() added
  - Nxt.blacklistPeer() added
  - Nxt.getBlocks() added
  - Nxt.setLogging() added
  - Utils.nqtToString() added

Version 1.5.0
  - EventRegister and EventWait support
  - Block.getBaseTarget() parsing error
  - Block.getBlockIdString() added
  - Nxt.getBalance() added
  - Nxt.getForging() added
  - Peer.setBlacklisted() added

Version 1.4.0
  - GetLog and GetStackTraces support

Version 1.3.0
  - SSL support
